Gmail API Setup

To enable real inbox data you need a Google OAuth Client ID.

  1. Go to console.cloud.google.com and create a project.
  2. Enable the Gmail API for your project.
  3. Under Credentials, create an OAuth 2.0 Client ID (Web Application).
  4. Add your deployed URL as an Authorized JavaScript Origin.
  5. Copy the Client ID and paste it into js/auth.js at the top.

About InboxDue

InboxDue connects to your real Gmail inbox via Google OAuth and surfaces what actually needs your attention — no more scanning every message manually.

  • Needs Reply — emails where a response is expected
  • Has Deadline — time-sensitive messages
  • Action Items — emails asking you to do something

All classification runs in your browser. InboxDue never sends, modifies, or deletes any email.
